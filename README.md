# СРОСС® — Сейсмобезопасность России

> Информационная система **Сейсмобезопасность России** (СРОСС®) — международный проект по устойчивому развитию, объединяющий данные о сейсмической опасности, методологию оценки рисков и инструменты целевого планирования градостроительной деятельности.

🌐 **Веб-сайт:** [seismos.ru](https://seismos.ru) (в разработке)
🏢 **Координатор:** Евразийская СЕЙСМО Ассоциация (ЕАСА)
📰 **Статус СМИ:** ЭЛ № ФС 77–39562 от 26.04.2010 (Роскомнадзор)
📜 **История:** развивается с 2009 года, преемственность научной школы с 1974 г.

## ✨ Возможности сайта

### 🎨 Брендинг
- **Эмблема СРОСС®** — круглый медальон с выпуклой металлической оправой (radial gradient) и объёмной линзой в центре
- **3 варианта оправы:** Soft / Medium / Strong — для выбора оптимального контраста
- **Дзен-анимация:** пульсирующие концентрические кольца от эпицентра (цикл 9 сек, ±3% амплитуда)
- **Топографический фон-карта** с розой ветров и координатами Москвы

### 🔊 Медитативный эмбиент
- Web Audio API синтез: 4 sine-осциллятора (55, 110, 165, 220 Гц) + LFO-модуляция + реверберация
- Кнопка включения/выключения с пульсирующим кольцом при активации
- Синхронизация «дыхания» звука с анимацией эмблемы

### 📋 Содержание
1. **Hero** — миссия, метрики, 3 варианта эмблемы, кнопка звука
2. **О системе** — статус СМИ, ЕАСА, хронология 1974→2026, цели ООН SDG 11
3. **17 направлений** — сейсмоизоляция, виброгашение, мониторинг, испытания и т.д.
4. **Партнёры** — 8 организаций + 9 регионов ЕАЭС
5. **Публикации** — 4 научные статьи 2004–2017 с PDF-ссылками
6. **Контакты** — форма с темами запросов + реквизиты СМИ

## 🚀 Установка и запуск

```bash
# Установка зависимостей
bun install

# Dev-сервер (sandbox-режим, карта отключена)
bun run dev

# Production build
NODE_OPTIONS="--max-old-space-size=512" bun run build

# Запуск production-сервера
NODE_OPTIONS="--max-old-space-size=256" bun run start
```

### Переменные окружения

Скопируйте `.env.example` в `.env`:

```bash
cp .env.example .env
```

| Переменная | По умолчанию | Описание |
|------------|--------------|----------|
| `NEXT_PUBLIC_MAP_ENABLED` | `false` | Включить интерактивную карту сейсмической опасности (требует ~847 КБ GeoJSON) |

## 🗺 Карта сейсмической опасности

Карта временно отключена для стабильности dev-сервера. Для включения в production:

```bash
# .env
NEXT_PUBLIC_MAP_ENABLED=true
```

Карта использует lazy-loading (IntersectionObserver) — 847 КБ GeoJSON не блокируют initial page load.
